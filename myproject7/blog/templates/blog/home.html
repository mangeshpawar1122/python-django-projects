{% comment %} this is a firstof {% endcomment %}

Firstof statement

Firstof statement is used to display the first non-empty value from a list of variables. It is useful when you want to display a default value if all other values are empty.

First non-empty blog title:

Author: {% firstof blog.1.author "deepak" %}

{% comment %} this is a verbatim statement {% endcomment %}

Verbatim statement

Verbatim statement is used to include a block of code that should not be processed by the template engine. It is useful when you want to include code that contains template syntax without it being interpreted.

Example of verbatim statement:

{% verbatim %}

This is a verbatim block. The following code will not be processed by the template engine:

{{ This will not be rendered }}

Author: {{ blog.1.author }}

{% endverbatim %}
{% comment %} this is a autoescape on/off statement {% endcomment %}

Autoescape statement

Autoescape statement is used to control the autoescaping of variables in the template. It is useful when you want to include HTML content in your template without it being escaped.

Example of autoescape statement:

This is an autoescape on block. The following code will be escaped:

We will use the following HTML code to demonstrate autoescaping:

default(Safe Escaped):{{ html_code }}

autoescape on : {{ html_code }}

Now we will turn off autoescape to see the difference:

autoescape off : {{ html_code|safe }}

{% autoescape off %}

This is an autoescape off block. The following code will not be escaped:

Autoescape Off :{{ html_code }}

{% endautoescape %}

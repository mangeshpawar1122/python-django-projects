width ratio is a template tag that calculates a ratio of two numbers and returns it as a percentage. The syntax is syntax: widthratio this_value max_value max_width . For example, if you want to calculate the percentage of 50 out of 100, you can use ex: widthratio 50 100 100 , which will return 50%.

Spaceless Example

Spaceless is a template tag that removes whitespace between HTML tags. It can be used to optimize the output of your templates by reducing the amount of unnecessary whitespace. The syntax is spaceless ... endspaceless . Any HTML code placed between these tags will have its whitespace removed.

outline filter

Outline filter is a template filter that adds an outline to text. It can be used to make text stand out and improve readability. The syntax is value|outline . For example, if you want to add an outline to the text "Hello World", you can use ex: "Hello World"|outline , which will return the text with an outline.

{% filter upper %}

This text will be converted to uppercase.

{% endfilter %}

inline filters

Inline filters are a way to apply filters directly to variables in your templates. They allow you to modify the output of variables without having to use the filter tag. The syntax is variable|filter . For example, if you have a variable called name and you want to convert it to uppercase, you can use ex: name|upper , which will return the value of name in uppercase.
